Memwiki Operations

Use the CLI through uv run memwiki during development.

uv run memwiki --workspace examples/basic init
uv run memwiki --workspace examples/basic ingest examples/basic/source.txt
uv run memwiki --workspace examples/basic ingest examples/basic/source.txt --dry-run
uv run memwiki --workspace examples/basic compile <source-id>
uv run memwiki --workspace examples/basic promote <draft-id> --check-only
uv run memwiki --workspace examples/basic promote <draft-id>
uv run memwiki --workspace examples/basic lint
uv run memwiki --workspace examples/basic query "HTML-first memory" --json
uv run memwiki --workspace examples/basic resolve <object-id>
uv run memwiki --workspace examples/basic backlinks <object-id>
uv run memwiki --workspace examples/basic capabilities
uv run memwiki --workspace examples/basic docs check
